# library-management-system

This system manages books and members.It tracks borrowed books and returned books,adds new books and registers new members with the help of the library membership ID.


## Features

 * ** Register new members:** registers new member by gigivng them a unique mebership ID

 * ** Library state:** Shows state of the books in the library as either available or borrowed 

 * ** Borrow book: ** the number of the book borrowed is recorded against the  library membership number and name 

 * ** Return Book :** the number of the book returned is recorded against the library membership number and name
